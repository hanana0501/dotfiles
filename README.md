## HOW TO

```

$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
$ brew tap Homebrew/bundle
$ brew bundle
$ PYTHON_CONFIGURE_OPTS="--enable-framework" pyenv install 3.6.0
$ pyenv global 3.6.0
$ pyenv rehash
$ mkdir $HOME/src
$ mkdir $HOME/bin

```
